Python 基础

课程简介：本课程讲解Python基础语法、变量、流程控制、函数、容器等核心知识点，是数据分析必备入门语言，侧重编程思维与基础实操训练。

题目1：基础输出

使用print函数在控制台输出“数据分析学习进行中”。

题目2：变量与数据类型

定义字符串、整型、浮点型变量，并分别打印变量及对应数据类型。

题目3：分支结构if判断

接收一个数字，判断该数字是正数、负数还是零。

题目4：for循环遍历列表

创建课程名称列表，使用循环逐行输出所有课程名。

题目5：while循环累加计算

利用while循环计算 1 到 50 所有整数的和。

题目6【代码运行题】：自定义函数实现两数运算

编写函数，接收两个参数，实现求和运算并调用函数输出结果。

题目7：列表增删改查

对列表执行追加、删除、修改元素操作，打印最终列表。

题目8：字典数据操作

创建个人信息字典，完成取值、修改、新增键值对操作。

题目9：字符串处理

对字符串进行切片、去除空格、大小写转换操作。

题目10：异常捕获

使用 try-except 捕获除数为0的运行异常。
